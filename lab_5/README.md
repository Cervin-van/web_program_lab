# 🚀 Personal CV Portfolio & Web Development Labs

Цей репозиторій містить вихідний код мого вебдодатка-резюме (CV), створеного в рамках вивчення курсу «Технології веб-програмування». Проєкт успішно розгорнуто на платформі Vercel.

## 🔗 Живий перегляд (Live Demo)

🌐 **[Переглянути сайт онлайн](https://web-program-lab.vercel.app)** ## 🛠 Технології та Інструменти (Tech Stack)

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla, ES6+).
- **API:** Fetch API (інтеграція з JSONPlaceholder).
- **Версіонування:** Git & GitHub.
- **Розгортання (Деплой):** Vercel.

## 💡 Реалізований функціонал

- Семантична та адаптивна розмітка за допомогою HTML та CSS (Flexbox).
- Інтерактивні елементи на JavaScript: модальне вікно форми зворотного зв'язку (через Formspree) та зчитування/запис системних даних у `localStorage`.
- Перемикач світлої/темної теми з автоматичним визначенням за часом доби.
- Асинхронне завантаження та рендеринг відгуків з віддаленого REST API.
